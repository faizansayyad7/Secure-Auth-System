# 🔐 Secure Authentication System with TOTP (2FA)

A web-based cybersecurity project built using Flask and Python that implements secure authentication with password hashing and Time-based One-Time Password (TOTP) verification. The application includes user registration, login workflows, QR code provisioning, login activity monitoring, and a cyber-themed security dashboard.

---

## 🚀 Features

✅ User Registration

✅ Secure Login System

✅ Password Hashing

✅ Two-Factor Authentication (TOTP)

✅ QR Code Generation for Authenticator Apps

✅ Session Management

✅ Login Activity Tracking

✅ Security Alerts

✅ Dashboard Statistics

* Total Logins
* Failed Attempts
* 2FA Status
* Recent Login Activity

✅ RGB Cyber-Themed UI
